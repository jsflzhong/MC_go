## 项目结构
```
-goPOC
    |-common 公共方法目录
    |-config 配置文件目录
    |-controller 控制器目录
    |-database 数据库访问目录
    |-model 数据库POJO目录
    |-foundation 基础知识&算法目录
    |-static 静态资源目录
        |-css css文件目录
        |-images 图片目录
        |-js js文件目录
        |-libs js类库
    |-system 系统配置文件加载目录
    |-test 测试目录
    |-view 模板文件目录
    |-pkg IDE在编译后自动生成的
    |-main.go 程序执行入口
```
